# Purple Wave Typing Speed Test 💜

This is my first project created in collaboration with AI.
Exploring coding, learning new things, and building ideas with AI assistance.

## About
A purple themed typing speed test made for ARMY.
It checks your typing speed in WPM and accuracy in real time.

## Features
- Live WPM counter
- Accuracy percentage 
- Timer in seconds
- BTS / ARMY theme with purple design

## How to use
1. Start typing the given text
2. Watch your WPM and Accuracy update live
3. Finish to see your final score
